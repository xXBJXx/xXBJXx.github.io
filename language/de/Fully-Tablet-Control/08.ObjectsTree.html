# Objects

  • bei den erstellten Datenpunkten kann man die Helligkeit manuell einstellen dazu muss man zuerst bei punkt 1 von Admin auf User wechseln, dann kann man bei punkt 3 die Helligkeit manuell verändern.

  • unter Punkt 2 wird bei einer Warnung, wenn das Tablet nicht auflädt ein true wert gesetzt.

  • bei punkt 4 wird der aktuelle Batterie stand für eine vis Anzeige geschrieben z.B. [ basic-Image 8 Widget] beim normalen betrieb, wird nur die % zahl geschrieben z.B. 10 % wen das tablet am Laden ist, wird charge 10 % angezeigt.
