    # Adapter Beschreibung

    Der Adapter liest die Wallpenel app Android (opens new window) aus und trägt die Werte in Datenpunkte ein.
    Man kann auch ein paar Funktionen steuern, wie z.B. die Helligkeit.
    Es können mehrere Devices abgefragt bzw. gesteuert werden.
